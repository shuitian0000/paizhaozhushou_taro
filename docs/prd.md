# 智能摄影助手微信小程序需求文档

## 1. 小程序概述

### 1.1 小程序名称
智能摄影助手

### 1.2 小程序描述
一款基于AI技术的智能摄影辅助工具，通过实时分析拍摄场景和人物姿态，为用户提供专业的摄影构图建议和评分，帮助用户快速提升摄影水平，拍出更出色的照片。\n
## 2. 核心功能

### 2.1 拍照助手模式
- 调用手机相机，全屏显示取景画面
- 每隔2秒自动采集镜头画面并进行实时分析
- 实时提供简短建议（每个维度不超过10个字）：\n  - 拍摄角度调整建议
  - 拍摄距离调整建议
  - 机位高度调整建议
  - 人物pose调整建议
- 实时为当前镜头画面打分（0-100分）\n- 显示简短的打分依据和改进方向

### 2.2 照片评估模式
- 用户上传已拍摄照片
- 系统自动分析照片
- 输出照片综合评分（0-100分）\n- 在分数附近显示简短的总体建议
- 提供详细评估报告：\n  - 构图分析
  - 人物姿态评估
  - 角度距离评价
  - 改进建议

## 3. 评价体系

### 3.1 评分维度
- 构图合理性（30%）：基于三分法、黄金螺旋等构图理论
- 人物姿态（30%）：姿态自然度、表情状态
- 拍摄角度（20%）：角度新颖性、主体突出度
- 拍摄距离（10%）：主体清晰度、环境协调度
- 机位高度（10%）：视角选择合理性

### 3.2 评分标准一致性
- 不同功能模式下采用统一的评价维度和权重
- 算法模型统一，确保线上线下评估结果一致性
- 针对不同场景类型（人像、风景、合影等）预设专项评估规则

## 4. 离线能力
- 核心评估算法本地化部署，支持弱网络或无网络环境使用
- 离线模式下保留完整功能体验，评分准确度不降低
- 离线状态提示：当前为离线评估模式，结果仅供参考

## 5. 用户交互

### 5.1 拍照助手界面
- 全屏取景窗口
- 实时分数显示区域
- 简短建议提示框（浮层展示）\n- 拍照按钮
- 切换前后置摄像头按钮
- 闪光灯开关

### 5.2 照片评估界面
- 照片预览区域
- 评分结果展示（含附近简短建议）\n- 评估报告详情
- 保存建议报告功能
- 未选择照片时，在选择照片按钮下方添加返回按钮

## 6. 技术要求

### 6.1 离线能力实现
- 核心评估算法本地化，支持离线推理
- 离在线状态自动切换
- 离线模式下性能优化，确保实时性

### 6.2 实时性要求
- 拍照助手模式下，建议延迟不超过200ms
- 界面流畅，无卡顿现象
- 评估结果即时呈现

### 6.3 多场景适配
- 支持人像摄影、风景摄影、合影等多种场景
- 根据场景自动切换评估侧重维度
- 提供针对性的专业建议

### 6.4 错误处理
- 分析失败时，显示具体错误原因（如网络异常、算法模型加载失败、相机权限问题等）\n- 提供重试按钮和解决方案提示
- 优化相机权限申请流程，确保顺利获取相机权限
- 增加相机状态检测，确保相机正常工作
- 优化算法模型推理流程，提升成功率和稳定性
