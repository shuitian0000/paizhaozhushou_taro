# 拍Ta智能摄影助手微信小程序需求文档

## 1. 小程序概述

### 1.1 小程序名称
拍Ta智能摄影助手

### 1.2 小程序描述
一款基于AI技术的智能摄影辅助工具，通过实时分析拍摄场景和人物姿态，为用户提供专业的摄影构图建议和评分，帮助用户快速提升摄影水平，拍出更出色的照片。\n
## 2. 核心功能

### 2.1 账号系统
- 支持微信授权登录
- 用户登录后，所有历史记录与账号关联
- 未登录用户可使用全部功能，但不保存任何历史记录
- 未登录时，点击历史记录标签，提示用户登录
- 登录用户可在我的页面查看已登录的账号和头像
- 未登录时显示默认空白头像，并标识未登录状态
- 未登录情况下点击头像可触发登录提示
- 登录用户可在我的页面点击退出账号，实现退出当前账号功能

### 2.2 拍照助手模式
- 调用手机相机，全屏显示取景画面
- 每隔2秒自动采集镜头画面并进行实时分析
- 实时提供简短建议（每个维度不超过10个字）：\n  - 拍摄角度调整建议
  - 拍摄距离调整建议
  - 机位高度调整建议
  - 人物pose调整建议
- 实时为当前镜头画面打分（0-100分）\n- 显示简短的打分依据和改进方向
- 登录用户可保存当前镜头画面及分析结果到历史记录

### 2.3 照片评估模式
- 用户上传已拍摄照片
- 系统自动分析照片
- 输出照片综合评分（0-100分）\n- 在分数附近显示简短的总体建议
- 提供详细评估报告：\n  - 构图分析
  - 人物姿态评估
  - 角度距离评价
  - 改进建议
- 登录用户可保存评估结果到历史记录

### 2.4 历史记录模块
- 登录用户可查看所有历史记录
- 展示已保存的镜头画面、评估分数、建议内容
- 支持查看历史记录的详细评估报告
- 未登录用户点击历史记录标签时，提示：请先登录微信账号，即可查看历史记录和使用全部功能
- 历史记录模块从首页移至我的页面

### 2.5 我的页面
- 上部显示已经登录的账号和头像，未登录情况下显示默认空白头像，并标识未登录
- 未登录情况下点击头像可触发登录提示
- 包含历史记录条目，跳转到历史记录页面
- 包含建议和吐槽条目
- 包含退出账号功能，实现退出当前账号

## 3. 评价体系

### 3.1 评分维度
- 构图合理性（30%）：基于三分法、黄金螺旋等构图理论
- 人物姿态（30%）：姿态自然度、表情状态
- 拍摄角度（20%）：角度新颖性、主体突出度
- 拍摄距离（10%）：主体清晰度、环境协调度
- 机位高度（10%）：视角选择合理性

### 3.2 评分标准一致性
- 不同功能模式下采用统一的评价维度和权重
- 算法模型统一，确保线上线下评估结果一致性
- 针对不同场景类型（人像、风景、合影等）预设专项评估规则

## 4. 离线能力
- 核心评估算法本地化部署，支持弱网络或无网络环境使用
- 离线模式下保留完整功能体验，评分准确度不降低
- 离线状态提示：当前为离线评估模式，结果仅供参考

## 5. 用户交互

### 5.1 首页布局
- 去掉顶部的总评估、平均分、实时拍摄布局条目
- 保留核心功能入口：拍照助手、照片评估
- 删除历史记录条目
- 底部导航栏增加我的页签

### 5.2 拍照助手界面
- 全屏取景窗口
- 实时分数显示区域
- 简短建议提示框（浮层展示）\n- 拍照按钮
- 切换前后置摄像头按钮
- 闪光灯开关

### 5.3 照片评估界面
- 照片预览区域
- 评分结果展示（含附近简短建议）\n- 评估报告详情
- 保存建议报告功能
- 未选择照片时，在选择照片按钮下方添加返回按钮

### 5.4 历史记录界面
- 展示历史记录列表
- 显示每条记录的评分、时间、场景类型
- 点击记录可查看详细评估报告
- 未登录时，点击历史记录标签提示登录

### 5.5 我的页面
- 上部显示用户信息区域（头像、昵称）\n- 历史记录条目，跳转到历史记录页面
- 建议和吐槽条目
- 退出账号条目

## 6. 技术要求

### 6.1 账号系统集成
- 实现微信授权登录功能
- 登录状态管理
- 用户信息存储与同步
- 历史记录与账号关联
- 完善微信登录页面，确保用户协议和隐私政策可点击查看对应条款

### 6.2 离线能力实现
- 核心评估算法本地化，支持离线推理
- 离在线状态自动切换
- 离线模式下性能优化，确保实时性

### 6.3 实时性要求
- 拍照助手模式下，建议延迟不超过200ms
- 界面流畅，无卡顿现象
- 评估结果即时呈现

### 6.4 多场景适配
- 支持人像摄影、风景摄影、合影等多种场景
- 根据场景自动切换评估侧重维度
- 提供针对性的专业建议

### 6.5 错误处理
- 分析失败时，显示具体错误原因（如网络异常、算法模型加载失败、相机权限问题等）\n- 提供重试按钮和解决方案提示
- 优化相机权限申请流程，确保顺利获取相机权限
- 增加相机状态检测，确保相机正常工作
- 优化算法模型推理流程，提升成功率和稳定性

### 6.6 用户协议和隐私政策
- 完善微信登录页面，确保用户协议和隐私政策可点击查看对应条款
- 用户协议内容应包含：服务条款、用户权利、服务使用规则、知识产权等
- 隐私政策内容应包含：个人信息收集范围、使用方式、存储保护、用户权利等
